# シフト作成ルール

1. **夜勤配置** —
   1. 毎日2人夜勤を配置
   2. 毎日1人のリーダーを夜勤に割り当て、もう1人はリーダー候補でない人を優先して組んでいく（ソフト条件）

2. **日勤配置**
   1. 日勤の定義は日と7b
   2. リーダー配置 —毎日 1人のリーダーが必要
   3. メンバー配置** — 以下の必要人数を満たすよう日勤を配置

      1. 平日は日が4人,7bが1人
      2. 日曜は、3回が日が5人、4階が日が4人
3. **夜勤3日セット** — 夜勤 → 明け → 公休



## 設定項目

| 設定                 | 説明                                 | デフォルト |
| -------------------- | ------------------------------------ | ---------- |
| 日勤リーダー可能人数 | 上から何人目までが日勤リーダー可能か | 10         |
| 夜勤リーダー可能人数 | 上から何人目までが夜勤リーダー可能か | 8          |
| 夜勤可能人数         | 上から何人目までが夜勤可能か         | 17         |
| 1日の日勤必要人数    | 日勤に必要な人数                     | 5          |
| 月間夜勤上限         | 1人あたりの月間夜勤回数上限          | 5          |
| 公休日数             | 1人あたりの月間公休日数              | 9          |





### 連続勤務制限：

日勤のみスタッフは5連勤まで

夜勤ありスタッフは4連勤まで、ただし★の前は３日勤まで



以下の数字はシートから取得

- 日勤リーダー可能人数 （上から）

- 夜勤リーダー可能人数 （上から）

- 夜勤可能人数 （上から）

- 1日の日勤必要人数月間

- 夜勤上限、

- 公休：シートから取得　公休日数：2月のみ8日、他の月は9日



事前に記入してある文字の変換

休：公休

★：夜勤リーダー
☆：夜勤

 ～、～⋆：夜勤明け

上記以外は日勤に変換する
